Samsung E-Paper App 功能確認報告

針對 Samsung E-Paper 應用程式之裝置加入、內容傳送、播放清單、排程與裝置設定進行功能整理
裝置：EM32DX-A 018E
Firmware：S-RSEDWWC-1100.0
內容模式：Contents / Playlist / Schedule
操作手冊：已附連結
1 確認結論

依據實測畫面與功能操作確認，Samsung E-Paper App 可作為本地端近場管理工具，用於加入機台、傳送單張圖片、建立多張圖片輪播、設定本地端排程，以及進行裝置顯示、電源、網路、安全與維護設定。

Add Device
QR Code / 藍牙搜尋
可掃描機台 QR Code，或透過 Scan nearby 搜尋附近裝置。
Content
單張 / 多張 / 排程
Contents、Playlist、Schedule 分別對應單張傳送、輪播清單、本地端排程。
Settings
完整裝置管理
包含顯示、睡眠、電源、憑證、韌體、網路與重置功能。
2 裝置加入方式
方式確認內容適用情境
QR Code 掃描App 畫面提示可掃描裝置螢幕上顯示的 QR Code 來加入機台。快速綁定指定單一裝置。
藍牙 / Nearby 搜尋可點選 Scan nearby，從附近裝置清單中搜尋與加入機台。現場多台設備或不方便掃描 QR Code 時使用。
3 內容操作模式
Contents
單張圖片傳送
較接近單張圖片內容管理，可將圖片匯入 App 後設定到裝置顯示。
→
Playlist
多張圖片輪播
可加入多張圖片，也可設定單張圖片的輪播時間與播放順序。
→
Schedule
本地端排程
可生成本地端播放排程，依指定時間播放內容或播放清單。
目前確認 App 底部有 Devices 與 Contents 入口；Contents 頁面內包含 Contents、Playlist、Schedule 三個分頁，可支援由單張內容到排程播放的操作流程。
操作限制：本地端排程需要針對每一張圖片逐一進行時間設定，若圖片數量較多，設定流程會比較繁瑣，現場大量內容更新時操作效率較低。

5 建議測試流程
  • 裝置加入：分別驗證 QR Code 掃描與 Scan nearby 搜尋是否可成功加入同一台機台。
  • 單張圖片：使用 Contents 上傳一張圖片並 Set content，確認裝置畫面更新結果。
  • 多張輪播：建立 Playlist，加入 2 張以上圖片，設定每張顯示時間後確認輪播是否符合設定。
  • 本地排程：建立 Schedule，確認排程時間到達後是否自動切換內容或播放清單；同時記錄一張圖一張圖設定時間所需的操作步驟與耗時。
  • 裝置設定：驗證 Screen Orientation、Picture Mode、Sleep After、Button Lock、USB Lock 等設定是否會同步到裝置。
